# Luke Barbas - Website Responsivo

Website moderno e responsivo para barbearia, criado com HTML5, CSS3 e JavaScript.
Foco em layout limpo, experiência do usuário e facilidade de contato.

## Tecnologias
- HTML5
- CSS3
- JavaScript (ES6)
- Template: HTML5 UP

## Preview
Veja o site online: [Clique aqui] (https://lucaareas.github.io/barbershop-site/)

## Como usar
1. Clone o repositório:
   git clone https:github.com/LucaAreas/barbershop-site.git
2. Abra o arquivo index.html no navegador.
